# # SPARK-CHALLENGE

<h2>To run the app</h2>

In the terminal input the following commands one by one

mvn clean
mvn compile
mvn assembly:single

java -cp target/spark-challenge-1.0.0-jar-with-dependencies.jar edu.nwmissouri.aksharagurram.App "data.txt"
